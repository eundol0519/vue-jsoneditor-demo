# 📋 Page Option Editor 기능 점검 리스트

## 1. Feature 관리 기능

### 1.1 Feature 추가
- [ ] "➕ Feature 추가" 버튼 클릭
- [ ] 입력 모달이 나타나는지 확인
- [ ] Feature ID 입력 (예: TEST001)
- [ ] "추가" 버튼 클릭
- [ ] Feature가 목록에 추가되는지 확인
- [ ] 성공 메시지가 표시되는지 확인
- **예상 결과**: Feature 목록에 새 항목이 추가됨

### 1.2 Feature 선택
- [ ] Feature 카드 클릭
- [ ] 카드가 활성화 상태(보라색)로 변경되는지 확인
- [ ] Option 목록이 해당 Feature의 Option들로 업데이트되는지 확인
- **예상 결과**: 선택된 Feature가 보라색으로 표시되고 Option 목록이 업데이트됨

### 1.3 Feature 삭제
- [ ] Feature 카드의 🗑️ 버튼 클릭
- [ ] 삭제 확인 다이얼로그가 나타나는지 확인
- [ ] "삭제" 버튼 클릭
- [ ] Feature가 목록에서 제거되는지 확인
- [ ] 성공 메시지가 표시되는지 확인
- **예상 결과**: Feature가 목록에서 삭제됨

---

## 2. Option 관리 기능

### 2.1 템플릿에서 Option 추가
- [ ] Feature 선택
- [ ] "➕ 템플릿에서 추가" 버튼 클릭
- [ ] 템플릿 선택 모달이 나타나는지 확인
- [ ] 템플릿 검색이 작동하는지 확인
- [ ] 템플릿 클릭
- [ ] Option이 추가되는지 확인
- [ ] value가 list[0].listValue로 설정되는지 확인
- **예상 결과**: 선택한 템플릿의 Option이 Feature에 추가됨

### 2.2 중복 Option 추가 시도
- [ ] 이미 존재하는 Option 템플릿 선택
- [ ] 경고 메시지가 표시되는지 확인
- [ ] "이미 사용 중인 옵션입니다" 메시지 확인
- **예상 결과**: 추가 불가 + 경고 알림

### 2.3 Option 선택
- [ ] Option 카드 클릭
- [ ] 카드가 활성화 상태(핑크색)로 변경되는지 확인
- [ ] Option 상세 편집 영역에 데이터가 표시되는지 확인
- [ ] 첫 번째 클릭에서 바로 데이터가 보이는지 확인 (중요!)
- **예상 결과**: 선택된 Option이 핑크색으로 표시되고 상세 정보가 즉시 표시됨

### 2.4 Option 삭제
- [ ] Option 카드의 🗑️ 버튼 클릭
- [ ] 삭제 확인 다이얼로그가 나타나는지 확인
- [ ] "삭제" 버튼 클릭
- [ ] Option이 목록에서 제거되는지 확인
- **예상 결과**: Option이 삭제됨

## 3. Option 상세 편집 기능

### 3.1 Option Key 수정
- [ ] Option 선택
- [ ] Option Key 입력 필드 변경
- [ ] 포커스 해제 (blur)
- [ ] Key가 변경되는지 확인
- [ ] 성공 메시지 확인
- **예상 결과**: Option Key가 변경됨

### 3.2 설명(desc) 수정
- [ ] 설명 textarea에 입력
- [ ] 입력한 내용이 즉시 반영되는지 확인
- **예상 결과**: 설명이 실시간으로 업데이트됨

### 3.3 값(value) 수정
- [ ] 값 입력 필드에 입력
- [ ] 입력한 내용이 즉시 반영되는지 확인
- **예상 결과**: 값이 실시간으로 업데이트됨

### 3.4 리스트 항목 보기 (읽기 전용)
- [ ] 리스트 항목이 읽기 전용으로 표시되는지 확인
- [ ] 수정 불가능한 상태인지 확인
- [ ] 안내 메시지가 표시되는지 확인
- **예상 결과**: 리스트 항목이 회색 배경의 읽기 전용으로 표시됨

---

## 4. 템플릿 관리 기능

### 4.1 템플릿 관리 열기
- [ ] "📚 템플릿 관리" 버튼 클릭
- [ ] 모달이 나타나는지 확인
- [ ] 템플릿 목록이 표시되는지 확인
- [ ] 모든 템플릿이 접힌 상태로 시작하는지 확인
- **예상 결과**: 템플릿 관리 모달이 열림

### 4.2 템플릿 펼치기/접기 (아코디언)
- [ ] 템플릿 헤더 클릭
- [ ] ▶ 아이콘이 ▼로 변경되는지 확인
- [ ] 템플릿 상세 정보가 나타나는지 확인 ⭐ **중요!**
- [ ] 다시 클릭하면 접히는지 확인
- **예상 결과**: 템플릿이 펼쳐지고 설명, 기본값, 리스트 항목, 사용 위치가 표시됨

### 4.3 템플릿 검색
- [ ] 검색 입력 필드에 텍스트 입력
- [ ] 템플릿 목록이 필터링되는지 확인
- **예상 결과**: 검색어와 일치하는 템플릿만 표시됨

### 4.4 새 템플릿 만들기
- [ ] "➕ 새 템플릿 만들기" 버튼 클릭
- [ ] 템플릿 생성 모달이 나타나는지 확인
- [ ] 템플릿 Key, 설명 입력
- [ ] "➕ 항목 추가" 버튼으로 리스트 항목 추가
- [ ] listValue, listDesc 입력
- [ ] "템플릿 생성" 버튼 클릭
- [ ] 템플릿이 목록에 추가되는지 확인
- [ ] "커스텀" 배지가 표시되는지 확인
- [ ] localStorage에 저장되는지 확인
- **예상 결과**: 새 커스텀 템플릿이 생성되고 목록에 표시됨

### 4.5 템플릿 수정
- [ ] 템플릿의 "✏️ 수정" 버튼 클릭
- [ ] 수정 모달이 나타나는지 확인
- [ ] 기존 값이 채워져 있는지 확인 (desc, list 항목들)
- [ ] 템플릿 Key가 비활성화되어 있는지 확인
- [ ] 설명 수정
- [ ] 리스트 항목 수정 (추가/삭제/편집)
- [ ] "수정 완료" 버튼 클릭
- [ ] 템플릿이 업데이트되는지 확인
- [ ] 해당 템플릿을 사용하는 모든 Feature의 Option도 업데이트되는지 확인
- **예상 결과**: 템플릿과 사용 중인 모든 Option이 업데이트됨

### 4.6 템플릿 삭제
- [ ] 템플릿의 "🗑️ 삭제" 버튼 클릭
- [ ] 삭제 확인 다이얼로그가 나타나는지 확인
- [ ] 사용 중인 경우 경고 메시지가 표시되는지 확인
- [ ] "삭제" 버튼 클릭
- [ ] 템플릿이 목록에서 제거되는지 확인
- [ ] 사용 중이던 Feature의 Option도 삭제되는지 확인
- **예상 결과**: 템플릿과 관련 Option이 모두 삭제됨

## 5. 데이터 관리 기능

### 5.1 변경사항 저장
- [ ] 데이터 수정 (Feature/Option 추가, 수정 등)
- [ ] "💾 변경사항 저장" 버튼 클릭
- [ ] 확인 다이얼로그가 나타나는지 확인
- [ ] "저장" 버튼 클릭
- [ ] 성공 메시지가 표시되는지 확인
- **예상 결과**: 변경사항이 세션에 저장됨 (originalData 업데이트)

### 5.2 초기화
- [ ] 데이터 수정
- [ ] "↺ 초기화" 버튼 클릭
- [ ] 확인 다이얼로그가 나타나는지 확인
- [ ] "초기화" 버튼 클릭
- [ ] 모든 변경사항이 취소되는지 확인
- [ ] 원본 데이터로 되돌아가는지 확인
- **예상 결과**: 모든 변경사항이 취소되고 원본 데이터로 복원됨

### 5.3 JSON 내보내기
- [ ] "📥 JSON 내보내기" 버튼 클릭
- [ ] JSON 파일이 다운로드되는지 확인
- [ ] 파일명이 "pageOption_export.json"인지 확인
- [ ] 파일 내용이 올바른 JSON 형식인지 확인
- **예상 결과**: JSON 파일이 다운로드됨
